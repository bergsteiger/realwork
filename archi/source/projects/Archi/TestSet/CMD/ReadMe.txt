Утилитные командные файлы для тестов Арчи.

Файл arun.ini - ini-файл для arun.exe, чтобы можно было запускать тесты как юристы запускают Арчи, т.е. с 
копированием dll с сервера. Для Твери.
arun_M.ini - для Москвы.

Дополнительные условия:

На компьютере существует каталог "C:\Base".
Диск k: указывает на "\\archi\archi" - оттуда берется файлы от сборки.
Диск p: указывает на "\\archi2\archi" - оттуда береться готовая тестовая база.

Описание: 

ClearCopyBase.cmd - получение чистой копии базы.
ArchiTest.cmd - ежедневные тесты.
DocsUnlock.cmd - разблокировка докуменов в базе. В каталоге base должен быть подкаталог:
Test_Unlock
  garant\
    lock001.jor
  main\
    bb_log.hdn
    bb_log.hdr
Фактически происходит копирование в каталог testbase
RunArchiTest.cmd - общий модуль для файлов ArchiTest.cmd и SimpleArchiTest.cmd 
SimpleArchiTest.cmd - версия ежедневных тестов без копирования файлов из сборки (берет версию из под ног).
