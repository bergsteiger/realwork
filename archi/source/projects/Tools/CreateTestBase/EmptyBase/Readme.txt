Конвертация базы из EmptyBase.
EmptyBase - эталонная версия от Сулковскго Александра, передаваемая регионам. 
Используется для создания тестовой версии базы с небольшими модификациями - добавления пользователей, для работы с базой.
Обычно требуется только при переходе на новый формат базы.

Чтобы сделать EmptyBase.rar для использования с CreateTestBase нужно:
1. Распакавать EmptyBase_XXX.zip (где XXX - номер версии базы) в любой каталог (лучше в \EmptyBase).
2. Запустить admin2 на EmptyBase и импортировать список пользователей из TestBase.csv. 
3. Выставить для группы Администраторы нужные права.
4. Распаковать содежимое homes.zip в \EmptyBase. Это настройки пользователя ArchiTester.
5. Запакавать с помощью rar.exe каталог с базой в EmptyBase.rar
6. Подложить созданный файл под ноги CreateTestBase.exe 
7. Положить EmptyBase.rar и закоммитеть в CVS: w:\archi\source\projects\Tools\CreateTestBase\TestBase\

Важно!!! Вне зависимости от состава пользователей, среди них должен быть пользователь ArchiTester. В противном случае тесты выполняться не будут и результаты не будут записыватсья в К!
