Краткая инструкция по установке Словаря терминов RUP и Гаранта 6.

1. Откройте оболочку Толкового Словаря с \\TOM\PUB\GARANT\BUS&LAW\windic32.exe
2. В меню выберите Файл->Открыть и в диалоге укажите файл из директории 
   %CVSROOT%\requirements\glossary\DATA\bus&law.key.
3. В следующий раз по умолчанию программа откроет базу терминов RUP.

Чтобы была интеграция между Microsoft Word и Словарем:
1. Закройте Microsoft Word, если открыт.
2. Откройте директорию, где установлен Microsoft Office.
2. Скопируйте в директорию Office файл 
   %CVSROOT%\requirements\glossary\BIN\WinDic32.DLL.
3. Скопируйте в директорию Office\STARTUP файл 
   %CVSROOT%\requirements\glossary\BIN\WinDic32.DOT.
4. Запустите Word: на панели инструментов появилась кнопка Словаря. Кроме того,
   в меню, доступном по правой кнопке, появились соответствующие пункты. 
   Интеграция аналогична интеграции из Гаранта в Словарь - выделите часть текста
   и нажмите на кнопку перехода.
