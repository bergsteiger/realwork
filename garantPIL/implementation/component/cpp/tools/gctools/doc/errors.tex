\section{Ошибки первого прохода}

\error{Не могу открыть файл '{\it имя файла}'}
\status{ошибка}
\comment{Невозможно открыть на чтение файл перечисленный в списке 
файлов компиляции.}
\solution{Убедитесь, что указанный файл существует.}

\error{Топик без заголовка или имени не помечен как NODOC}
\status{ошибка}
\comment{Топик в указаной строке не кодержит короткого имени в команде
{\tt !TOPIC} и длинного в команде {\tt !NAME}.}
\solution{Внесите короткое имя в соответствующую команду {\tt !TOPIC} и/или
добавьте команду {\tt !NAME}}

\error{Повторяющийся номер топика {\it номер топика}}
\status{фатальная ошибка}
\comment{В массиве компилируемых топиков встречаются два топика с 
одинакомыми номерами.}
\solution{Измените номер одного из документов.}

\error{Неизвестный TYPE {\it название типа}}
\status{ошибка}
\comment{Встреченый в команде TYPE тип не указан в файле типов (см. документацию
по опциям компилятора).}
\solution{Измените тип документа или добавьте неизвестный тип в файл описания
типов}

\error{Повторяющийся тип {\it название типа}}
\status{ошибка}
\comment{В документе встрачаются две комманды TYPE с одинаковыми аргументами}
\solution{Удалите одну из команд}

\error{Неизвестный SOURCE {\it название источника}}
\status{ошибка}
\comment{Встреченый в команде SOURCE источник не указан в файле источников 
(см. документацию по опциям компилятора).}
\solution{Измените источник документа или добавьте неизвестный источник в 
файл описания источников (последнее могут делать только юристы центрального 
офиса)}

\error{Повторяющийся SOURCE {\it название источника}}
\status{ошибка}
\comment{В документе встрачаются две комманды SOURCE с одинаковыми аргументами}
\solution{Удалите одну из команд}

\error{Повторяющаяся команда PRIORITY}
\status{ошибка}
\comment{В документе встретилось две команды PRIORITY}
\solution{Удалите одну из них}

\error{Повторяющийся SUB.}
\status{ошибка}
\comment{в документе встретилось две комманды SUB с одинаковыми номерами}
\solution{Удалите или перенумеруйте одну из них}
