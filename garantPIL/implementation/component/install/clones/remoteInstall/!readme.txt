Исходники процедуры автообновления клонов.

Описание клонов: clones.xml
Описание xml: http://mdp.garant.ru/pages/viewpage.action?pageId=133892684

Для запуска:
1. откорректировать описание клонов
2. получить дистрибутив клона, которым будет обновляться существующие версии
3. положить полученный дистрибутив в распакованном виде в директорию distrib
4. убедиться, что в полученном на этапе (3) дереве директорий дистрибутива имеется директория scripts и файл AutoInstaller.py
5. запаковать полученный на этапе (4) дистрибутив в самораспаковывающийся архив distrb.exe так чтобы он был в директории distrib
6. в случае, если для какого-либо клона указана локальная установка (узел <localdistrib>true</localdistrib>) дистрибутив должен находиться в локальной директории c:\distrib\distrib.exe
7. запустить RemoteInstall.py



Применение патчей для набора клонов.

Для применения патчей используется xml описания клонов. Информация для каждого из них берется из секции old.
Патч полуавтоматический - необходимо указать в коде RemoteImport.py какие файлы импортировать.

Для запуска:

2. запустить RemoteImport.py (лучше логировать вывод - клонов много)
