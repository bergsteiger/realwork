Активность: Управлять проблемами

Цель
  • Инициация необходимых действий по проблемам, возникающим в ходе проекта
Шаги
  • Оценить проблемы
  • Определить действия по корректировке
  • Внести запросы на изменение и/или Назначить работы
Входные артефакты:
  • Оценка статуса
 Результирующие артефакты:
  • Запрос на изменение
Роль: Менеджер проекта

Оценка статуса создается регулярно и обнаруживает проблемы или "вещи, идущие неправильно". Это могут быть проблемы проекта (такие как отклонения от планов), Проблемы продукта (дефекты, неясность требований, технологичекие проблемы), и реализовавшиеся риски.

Менеджер проекта также инициирует эту активность для решения встающих проблем.

Оценить проблемы

Первый шаг - оценка проблемы. Для каждой проблемы определите ее влияние на проект, определите срочность ее решения.

Определить действия по корректировке

Корректировка может повлечь за собой необходимость изменения Общего плана проекта или требований, или проработки проектных решений. При этом необходимо создавать соответствующие Запросы на изменение.

Внести запросы на изменение и/или Назначить работы

Как для каждой из проблем определены решения, Менеджер проекта создает Запрос на изменение или Изменяет План проекта необходимым образом, передавая внесение корректив в продукт разработчикам.
