Цель

  • Внести статистическую информацию по проекту в базу сбора статистики
  • Изменить планы проекта в соответствии с уточненными данными
Входные артефакты: Результирующие артефакты:
  • Запросы участников, которые необходимо реализовать в итерации
  • Статус проекта
  • План итерации
  • План итерации (обновленный)

Роль: Менеджер проекта

Причины изменения планов

Изменения планов могут быть вызваны:

  • из-за ошибки планирования, которые определяются в ходе при отслеживании хода работ
  • добавления новых работ
  • уменьшения запланированных ресурсов в связи с форс-мажорными обстоятельствами

Внести статистическую информацию по проекту в базу сбора статистики

При проведении оценки проекта становится очевидным, что часть работ выполнена в соответствии с планом, часть - с опозданием, а часть - с опережением относительно намеченного плана. Менеджер проекта собирает эту информацию в базу данных и указывает затраченные ресурсы на проведение каждой из активностей.

Изменить планы проекта в соответствии с уточненными данными

Планы проекта должны быть изменены в соответствии с данными, полученными в ходе итерации проекта. Так, если имеется отставание, планы должны быть пересмотрены в сторону увеличения трудоемкости, если имеется опережение - то в сторону сокращения сроков разработки. См. также Активность: Распределить работы и ресурсы.

Менеджер проекта вносит соответствующие коррективы, согласовывает их с членами команды и утверждает новые планы проекта у Руководителя ОИТ.

После утверждения измененных планов, должны пересматриваться планы всех связанных групп (например, Группы качества), с последующим утверждением планов Руководителем ОИТ.

При ошибках планирования

После утверждения измененных планов, Менеджер проекта должен оповестить всех участников проекта об этих изменениях.

При добавлении новых работ

Менеджер проекта вносит в планы работы по анализу новых требований, проектированию и реализации

После утверждения измененных планов, Менеджер проекта должен оповестить всех участников проекта об этих изменениях.

При уменьшении запланированных ресурсов

Менеджер проекта возлагает ответственность за проведение работ, возложенных на эти ресурсы, на других сотрудников.

После утверждения измененных планов, Менеджер проекта должен оповестить всех участников проекта об этих изменениях.
