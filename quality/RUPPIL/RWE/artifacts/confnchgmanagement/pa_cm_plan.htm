Артефакт: План конфигурационного контроля


План конфигурационного контроля		 План конфигурационного контроля содержит все необходимые сведения по управлению конфигурациями в рамках проекта. План конфигурационного контроля дополняет Инструкцию конфигурационного контроля, вместе они содержат всю необходимую информацию для проведения активностей, связанных с конфигурационным управлением проекта.
Инструментарий: Microsoft® Word™
Шаблон: План конфигурационного контроля
Специальные сведения: План конфигурационного контроля, предлагаемый RUP, был разбит на 2 части: на Инструкцию конфигурационного контроля, действующую для всего ОИТ, и на План конфигурационного контроля, в котором прописаны сведения для конкретного проекта.
Связанные артефакты:
  • Инструкция конфигурационного контроля
  • План разработки ПО
  • Модель управления запросами на изменение
  • Инструкция тестирования
