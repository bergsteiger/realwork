Артефакт: Release  Notes


Release Notes		 Release Notes - это артефакт, предназначенный для пользователя программного продукта и содержащий краткую информацию о нововведениях в программном продукте.
Инструментарий: Microsoft® Word™
Шаблон: RUP : Шаблон Release Notes.
Специальные сведения: Release Notes не входят в состав архива установочной программы. Их можно менять вплоть до начала поставки. Они пишутся в текстовом формате в кодировке той ОС, для которой разработан программный продукт
Release Notes должны показываться по окончанию установки программного продукта (после выполнения функций артефактами инсталляции.
