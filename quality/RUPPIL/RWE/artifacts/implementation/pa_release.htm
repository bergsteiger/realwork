Артефакт: Выпуск


Выпуск		 Выпуск представляет собой законченный результат итерации разработки.
Инструменатрий: Средства разработки
Шаблон: Нет
Специальные сведения: Выпуски подразделяются на BUGFIX'ы и RELEASE'ы. BUGFIX предназначен для исправления ошибок, найденных со времени предыдущего выпуска. RELEASE - это версия,в которой реализована новая функциональность.
Все выпуски в обязательном порядке проходят комплексное тестирование
