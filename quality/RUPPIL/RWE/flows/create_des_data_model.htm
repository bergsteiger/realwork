Создать проектную модель и модель данных

В ходе этого подпотока должны быть спроектированы прецеденты, выбранные на итерацию, и доработана модель базы данных.

Активности:

  • Провести декомпозицию на подсистемы
  • Определить проектные элементы
  • Спроектировать прецедент
  • Спроектировать базу данных
  • Спроектировать подсистему
  • Провести обзор проектной модели
