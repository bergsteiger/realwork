Стереотипы UML

  • "Actor" [стереотип актера]
    Этот стереотип задается для любого элемента модели, описывающего актера взаимодействующего с системой. Общее правило стереотипизирования таких актеров - если актер управляет нашей системой (т.е. является ее пользователем), то элементу модели, описывающему такого актера, должен быть присвоен такой стереотип. Элементы модели, имеющие этот стереотип, могут располагаться только в пакетах, которые описывают прецедентное представление проектируемой системы.
