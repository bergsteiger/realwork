Фаза: Конструирование


План для фазы Конструирование

Описание

Фаза Конструирование - обычно наиболее продолжительная из всех фаз, на ней проводится наибольшее количество итераций. Перед началом работ на каждой итерации определяется список прецедентов, подлежащих реализации.

На фазе Конструирование проводится работа в рамках выбранных прецедентов:

  • В потоке работ Анализ и проектирование проводятся работы по созданию/проектированию необходимых артефактов в рамках Проектной Модели
  • Реализуется часть необходимых компонент, назначенная для реализации в ходе итерации
  • Созданные артефакты проходят тестирование в Группе Качества
  • По окончанию работ, проводятся работы в рамках потока работ Развертывание

Результат

Список важнейших артефактов, которые должны быть созданы на фазе Конструирование:
  • Проектная Модель
  • Модель реализации
  • Прототип реально работающей программы (системы)
