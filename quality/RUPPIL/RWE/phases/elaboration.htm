Фаза: Уточнение


План для фазы Уточнение

Описание

На Фазе Уточнение прорабатываются найденные прецеденты, формируется пользовательский интерфейс программы. Создается модель анализа, определяется архитектура системы, часть архитектурно-значимых прецедентов проектируется и реализуется.

Результат

Основная задача фазы Уточнение - создание модели анализа как основы для дальнейшего проектирования и выбор архитектуры системы.
Список важнейших артефактов, которые должны быть созданы на фазе Начало:

  • Software Requirements Specification (Модель прецедентов + дополнительные спецификации)
  • Прототип пользовательского интерфейса
  • Модель анализа
  • Документ программная архитектура
