Фаза:Начало

План итерации фазы Начало

Описание

На Фазе Начало выделяются заинтересованные лица по проекту, создается укрупненный план, собираются запросы участников, устанавливается среда проекта.

Результат

Основная задача фазы Начало - организовать проект, оценить его продолжительность и бюджет, собрать важнейшую информацию для построения системы.
Список основных артефактов, которые должны быть созданы на фазе Начало:

  • Видение
  • Запросы участников
  • Глоссарий
  • Список рисков
  • План проекта
Сам проект в случае решения его разработки должен быть одобрен высшим руководством, и на него должны быть выделены необходимые ресурсы.
