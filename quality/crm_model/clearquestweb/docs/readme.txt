
Простое веб-приложение для вывода предложений из базы ClearQuest. 

Сделано по просьбе УМ (Шер, Ригина) 03.07.2003 на основании документа
"Доступ к информации базы предложений.doc".

---------------------------------

Приложение работает с пользовательской базой данных GS (MS SQL сервер 
process). Его Функционирование возможно при условии неизменности 
схемы ClearQuest, на которой основана база GS. Если изменения в схеме 
затронут поля, к которым обращается приложение, то необходимо будет 
соответствующим образом модифицировать скрипты приложения.

В качестве веб-сервера используется IIS, ASP. Скриптовый язык - 
JScript.

---------------------------------
Для запуска приложения необходимы следующие компоненты.

1) Rational ClearQuest, установленный на той же машине, что и 
веб-сервер.

2) Веб-сервер IIS с виртуальной директорией "CQ", ключевые настройки 
которой перечисляются ниже:

    Права: read, write  
    Application name: cq
    Execute permissions: Scripts only
    Default document: default.asp
    Default ASP Language: JScript

Скрипты и каталоги следует разместить в данной виртуальной 
директории.

3) Если неправильно настроен IIS, то, возможно, потребуется выдать 
права записи в директорию "CQ" и все ее поддиректории встроенному 
пользователю IUSR_PROCESS, от имени которого IIS работает с файловой 
системой.
