В данном каталоге находятся различные утилиты для работы, пока с XML-ми
каталог <Perl>
<countusr> - подсчет количесва пользователей в файле <user.xml>
 countusr.bat - в качестве параметра можно передавать путь к XML файлу
		или просто запустить в каталоге с файлом <users.xml>
 countusr.pl

<userfltr> - проверяет соотвествие между наличием каталога с настройками пользователя
	     и записью в <uses.xml> о данном пользователе.
	     Если есть в XML файле пользователь "111" а каталога "111"  нет, то такие 
	     пользователи копируются в файл <uses.xml.bad>, если каталог и пользователь
	     есть, то "пользователь" копируется в файл <uses.xml.new>
	     По моему в названии каталогов и пользователей учитывается "регистр".

 userfltr.bat - параметров не требует, просто запустить в каталоге с файлом <users.xml>
	 	и папками с настройками прользователя
 userfltr.pl
