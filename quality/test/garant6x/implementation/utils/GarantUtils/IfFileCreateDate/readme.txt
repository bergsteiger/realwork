IfFileCreateDate.exe <filename> <mins>

Проверяет время модификации указанного файла.
Используется для автоответчика на \\strelka-vm1, c:\consultingserver.

Результат ErrorLevel:
0 - с момента модификации файла <filename> прошло менее <mins>
1 - с момента модификации файла <filename> прошло более <mins> или файл не существует
